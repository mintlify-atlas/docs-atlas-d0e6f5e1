## Overview

Laravel Films & Actors implements two custom middleware classes to validate and protect routes from invalid parameters. These middleware ensure data integrity and provide a better user experience by catching errors early.

<Note>
Both middleware classes validate year parameters but are applied to different route groups - `ValidateYear` for films and `ValidateURLActor` for actors.
</Note>

## Year Validation for Films

The `ValidateYear` middleware protects film routes from invalid year parameters.

### ValidateYear Middleware

```php ValidateYear.php
namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class ValidateYear
{
    /**
     * Handle an incoming request.
     *
     * @param  \Illuminate\Http\Request  $request
     * @param  \Closure(\Illuminate\Http\Request): (\Illuminate\Http\Response|\Illuminate\Http\RedirectResponse)  $next
     * @return \Illuminate\Http\Response|\Illuminate\Http\RedirectResponse
     */
    public function handle(Request $request, Closure $next)
    {
        $year = $request->route('year');

        // in case year is not numeric go to homepage
        if(isset($year)){
            if(is_null($year) || !is_numeric($year) || $year < 1900 || $year > 2024){
                  \Log::error("Invalid year range accessed in middleware: $year");
                  return redirect('/');
            }
        }

        return $next($request);        
    }
}
```

### Protected Routes

The middleware is applied to all film routes under the `filmout` prefix:

```php routes/web.php
Route::middleware('year')->group(function() {
    Route::group(['prefix'=>'filmout'], function(){
        Route::get('oldFilms/{year?}', [FilmController::class, "listOldFilms"])
            ->name('oldFilms');
        Route::get('newFilms/{year?}', [FilmController::class, "listNewFilms"])
            ->name('newFilms');
        Route::get('filmsByYear/{year?}', [FilmController::class, "listFilmsByYear"])
            ->name('filmsByYear');
        Route::get('filmsByGenre/{genre?}', [FilmController::class, "listFilmsByGenre"])
            ->name('filmsByGenre');
        Route::get('/countFilms', [FilmController::class, 'countFilms']);
        Route::get('/sortFilms', [FilmController::class, 'sortFilms']);
    });
});
```

## Actor URL Validation

The `ValidateURLActor` middleware protects actor routes with similar year validation logic.

### ValidateURLActor Middleware

```php ValidateURLActor.php
namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class ValidateURLActor
{
    /**
     * Handle an incoming request.
     *
     * @param  \Illuminate\Http\Request  $request
     * @param  \Closure(\Illuminate\Http\Request): (\Illuminate\Http\Response|\Illuminate\Http\RedirectResponse)  $next
     * @return \Illuminate\Http\Response|\Illuminate\Http\RedirectResponse
     */
    public function handle(Request $request, Closure $next)
    {
        $year = $request->route('year');

        // Check if year is valid if it is set
        if (isset($year)) {
            if (!is_numeric($year) || $year < 1900 || $year > 2024) {
                \Log::error("Invalid year range accessed in actor middleware: $year");
                return redirect('/');
            }
        }

        return $next($request);
    }
}
```

### Protected Routes

The middleware is applied to actor routes:

```php routes/web.php
Route::middleware('actor')->group(function() {
    Route::get('allActors', [ActorController::class, "listAllActors"])
        ->name('allActors');
    Route::get('actorsByDecade/{year?}', [ActorController::class, "listActorsByDecade"])
        ->name('actorsByDecade');
});
```


## Middleware Comparison

### Similarities

Both middleware classes share the same core validation logic:

  <Card title="Year Range" icon="calendar">

  <Card title="Numeric Check" icon="calculator">
    Must be numeric value
  </Card>
  <Card title="Redirect on Error" icon="right-from-bracket">
    Invalid requests go to `/`
  </Card>
</CardGroup>

### Differences

| Feature | ValidateYear | ValidateURLActor |
|---------|--------------|------------------|
| **Applied To** | Film routes (`filmout/*`) | Actor routes (`allActors`, `actorsByDecade`) |
| **Middleware Alias** | `year` | `actor` |
| **Null Check** | `is_null($year) \|\|` included | No explicit null check |
| **Error Log Message** | "Invalid year range accessed in middleware" | "Invalid year range accessed in actor middleware" |

<Note>
The slight difference in null checking doesn't affect behavior since both allow optional year parameters through `{year?}` route syntax.
</Note>

## Error Logging

Both middleware log validation failures to Laravel's error log:

```php
\Log::error("Invalid year range accessed in middleware: $year");
```

## Additional Validation

Beyond middleware, the application includes controller-level validation for film creation:

### Film Creation Validation

```php FilmController.php
// Year validation during film creation
$year = (int)$_POST['year'];
if ($year < 1900 || $year > 2024) {
    \Log::error("Attempted to add film with invalid year: $year");
    return view('welcome', [
        'error' => 'El año de la película debe estar entre 1900 y 2024'
    ]);
}
```

<Warning>
This provides **double validation**: middleware protects routes, and controllers validate data before saving.
</Warning>

### Duplicate Film Check

The film creation also validates against duplicates:

```php FilmController.php
foreach ($films as $film) {
    if (strcasecmp($film['name'], $_POST['name']) === 0) {
        return view('welcome', [
            'error' => 'La película ya existe en la lista'
        ]);
    }
}
```

## Middleware Registration

To use these middleware, they must be registered in `app/Http/Kernel.php`:

```php Kernel.php
protected $routeMiddleware = [
    // ... other middleware
    'year' => \App\Http\Middleware\ValidateYear::class,
    'actor' => \App\Http\Middleware\ValidateURLActor::class,
];
```

## Extending Validation

To add custom validation logic:

```php
// Example: Add genre validation
public function handle(Request $request, Closure $next)
{
    $year = $request->route('year');
    $genre = $request->route('genre');

    // Validate year
    if (isset($year)) {
        if (!is_numeric($year) || $year < 1900 || $year > 2024) {
            \Log::error("Invalid year: $year");
            return redirect('/');
        }
    }

    // Validate genre
    if (isset($genre)) {
        $validGenres = ['action', 'drama', 'comedy', 'thriller'];
        if (!in_array(strtolower($genre), $validGenres)) {
            \Log::error("Invalid genre: $genre");
            return redirect('/');
        }
    }

    return $next($request);
}
```
