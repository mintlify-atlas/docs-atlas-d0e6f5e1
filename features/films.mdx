## Overview

The Film Management system provides a complete solution for listing, filtering, sorting, and creating film entries. Films are stored in JSON format and can be filtered by year, genre, and other criteria.

<Note>
All film data is read from `films.json` in the controller directory. The application validates years between 1900 and 2024.
</Note>

## Core Methods

### Reading Films

The `readFilms()` static method reads all films from the JSON data source:

```php FilmController.php
public static function readFilms(): array
{
    $jsonFile = __DIR__ . '/films.json';

    if (!file_exists($jsonFile)) {
        return [];
    }

    $films = json_decode(file_get_contents($jsonFile), true);

    return is_array($films) ? $films : [];
}
```

### Listing All Films

Display all available films without any filtering:

```php FilmController.php
public function listAllFilms()
{
    $title = "Listado de todas las películas";
   
    $films = FilmController::readFilms();

    if (empty($films)) {
        $title = "No hay películas disponibles";
        return view("films.list", [
            "films" => [],
            "title" => $title
        ]);
    }

    return view("films.list", [
        "films" => $films,
        "title" => $title
    ]);
}
```

**Route:**
```php
Route::get('allFilms/{year?}/{genre?}', [FilmController::class, "listAllFilms"])
    ->name('allFilms');
```

**URL Example:**
- `/filmout/allFilms`

## Filtering Films

<Tabs>
  <Tab title="By Year">
    Filter films released in a specific year:

    ```php FilmController.php
    public function listFilmsByYear($year = null)
    {
        $films_filtered = [];
        $title = "Listado de películas por año";

        $films = FilmController::readFilms();

        if (is_null($year)) {
            $title = "Listado de todas las películas (sin filtrar por año)";
            return view("films.list", ["films" => $films, "title" => $title]);
        }

        foreach ($films as $film) {
            if ($film['year'] == $year) {
                $films_filtered[] = $film;
            }
        }

        $title = "Listado de películas del año $year";
        return view("films.list", ["films" => $films_filtered, "title" => $title]);
    }
    ```

    **Route:**
    ```php
    Route::get('filmsByYear/{year?}', [FilmController::class, "listFilmsByYear"])
        ->name('filmsByYear');
    ```

    **URL Examples:**
    - `/filmout/filmsByYear/2020` - Films from 2020
    - `/filmout/filmsByYear` - All films (no filter)
  </Tab>

  <Tab title="By Genre">


    **URL Examples:**

    - `/filmout/filmsByGenre/Action` - Action films (case-insensitive)

## Sorting and Counting

### Sort Films by Year

Sort all films by release year in descending order:

```php FilmController.php
public function sortFilms()
{
    $films_sort = [];
    $films = self::readFilms();

    usort($films, function($a, $b) {
        return $b['year'] <=> $a['year'];
    });

    $films_sort = $films;
    $title = "Peliculas ordenadas";
    return view("films.list", ["films" => $films_sort, "title" => $title]);
}
```

**Route:**
```php
Route::get('/sortFilms', [FilmController::class, 'sortFilms']);
```

**URL:** `/filmout/sortFilms`

### Count Films

Get the total number of films in the database:

```php FilmController.php
public function countFilms()
{
    $films = self::readFilms(); 
    $count = count($films);
    $title = "Total de películas";

    return view('films.count', ['count' => $count, 'title' => $title]);     
}
```

**Route:**
```php
Route::get('/countFilms', [FilmController::class, 'countFilms']);
```

**URL:** `/filmout/countFilms`

## Adding New Films

<Warning>
Before adding a film, the system validates:
- Film name doesn't already exist (case-insensitive)
- Year is between 1900 and 2024
- All required fields are provided
</Warning>

### Create Film Method

```php FilmController.php
public function createFile(Request $request)
{
    if ($_SERVER['REQUEST_METHOD'] === 'POST') {
        $uploadDir = __DIR__ . '/uploads/';
        $jsonFile = __DIR__ . '/films.json'; 

        if (!is_dir($uploadDir)) {
            mkdir($uploadDir, 0755, true);
        }

        $films = [];
        if (file_exists($jsonFile)) {
            $films = json_decode(file_get_contents($jsonFile), true);
        }

        // Check for duplicate film names
        foreach ($films as $film) {
            if (strcasecmp($film['name'], $_POST['name']) === 0) {
                return view('welcome', [
                    'error' => 'La película ya existe en la lista'
                ]);
            }
        }

        // Validate year range
        $year = (int)$_POST['year'];
        if ($year < 1900 || $year > 2024) {
            \Log::error("Attempted to add film with invalid year: $year");
            return view('welcome', [
                'error' => 'El año de la película debe estar entre 1900 y 2024'
            ]);
        }

        $newFilm = [
            'name' => $_POST['name'],
            'year' => $year,
            'genre' => $_POST['genre'],
            'country' => $_POST['country'],
            'duration' => $_POST['duration'],
            'img_url' => $_POST['img_url']
        ];

        $films[] = $newFilm;

        $saved = file_put_contents($jsonFile, json_encode($films, JSON_PRETTY_PRINT));

        if ($saved == false) {
            return view('welcome', ['error' => 'Error saving the film list']);
        }

        return $this->listAllFilms();
    }   
}
```

**Route:**
```php
Route::post('/add-film', [FilmController::class, 'createFile'])
    ->name('filmin.addFilm');
```

**URL:** `POST /filmin/add-film`

## Route Protection

Film routes under the `filmout` prefix are protected by the `ValidateYear` middleware:

```php routes/web.php
Route::middleware('year')->group(function() {
    Route::group(['prefix'=>'filmout'], function(){
        Route::get('oldFilms/{year?}', [FilmController::class, "listOldFilms"]);
        Route::get('newFilms/{year?}', [FilmController::class, "listNewFilms"]);
        Route::get('filmsByYear/{year?}', [FilmController::class, "listFilmsByYear"]);
        Route::get('filmsByGenre/{genre?}', [FilmController::class, "listFilmsByGenre"]);
        // ... more routes
    });
});
```

See the [Filtering](/features/filtering) page for details on year validation middleware.

## View Integration

Films are displayed using the `films.list` Blade template:

```blade films/list.blade.php
<h1>{{$title}}</h1>

@if(empty($films))

```
