## Overview

This endpoint returns actors filtered by their birth decade. You can specify a decade year (e.g., 1950, 1960, 1970) to get actors born within that 10-year period. If no year is provided, it returns all actors.

## Endpoint

```
GET /actorsByDecade/{year?}
```

## Middleware

This endpoint is protected by the `actor` middleware (`ValidateURLActor`) which validates the year parameter.

### Validation Rules

<Warning>
  If a year parameter is provided, it must meet these criteria:
  - Must be numeric
  - Must be greater than or equal to 1900
  - Must be less than or equal to 2024
  
  Invalid years will redirect to the home page (`/`) and log an error.
</Warning>

## Request

### Path Parameters

<ParamField path="year" type="integer" optional>
  The decade start year to filter actors by (e.g., 1950, 1960, 1970).
  
  - When provided: Returns actors born between `year` and `year + 9`
  - When omitted: Returns all actors (no filtering)
  - Must be between 1900 and 2024
  - Must be numeric
</ParamField>

### Headers

No special headers required.

## Implementation Details

### Controller Method

Location: `app/Http/Controllers/ActorController.php:52`

```php
public function listActorsByDecade($year = null)
{
    $decades = ActorController::getAvailableDecades();

    if (is_null($year)) {
        $title = "Listado de todos los actores (sin filtrar por década)";
        $actors = ActorController::readActors();
    } else {
        $endYear = $year + 9;
        $title = "Listado de actores nacidos en la década de $year a $endYear";
        $actors = Actor::whereBetween('birthdate', [$year, $endYear])
                      ->orderBy('birthdate', 'asc')
                      ->get()
                      ->toArray();
    }

    return view("actors.list", [
        "actors" => $actors, 
        "title" => $title, 
        "decades" => $decades,
        "selectedDecade" => $year
    ]);
}
```

### Database Query

When filtering by decade:

```php
Actor::whereBetween('birthdate', [$year, $year + 9])
     ->orderBy('birthdate', 'asc')
     ->get()
     ->toArray()
```

When no year is provided:

```php
Actor::orderBy('birthdate', 'asc')->get()->toArray()
```

### Available Decades Calculation

Location: `app/Http/Controllers/ActorController.php:18`

```php
public static function getAvailableDecades(): array
{
    $years = Actor::select('birthdate')
                  ->distinct()
                  ->orderBy('birthdate', 'asc')
                  ->pluck('birthdate')
                  ->toArray();
    $decades = [];
    foreach ($years as $y) {
        $decade = floor($y / 10) * 10;
        if (!in_array($decade, $decades)) {
            $decades[] = (int)$decade;
        }
    }
    return $decades;
}
```

### Middleware Validation

Location: `app/Http/Middleware/ValidateURLActor.php:17`

```php
public function handle(Request $request, Closure $next)
{
    $year = $request->route('year');

    if (isset($year)) {
        if (!is_numeric($year) || $year < 1900 || $year > 2024) {
            \Log::error("Invalid year range accessed in actor middleware: $year");
            return redirect('/');
        }
    }

    return $next($request);
}
```

## Response Behavior

### Without Year Parameter

```
GET /actorsByDecade
```

- Returns all actors (no filtering)
- Title: "Listado de todos los actores (sin filtrar por década)"
- `selectedDecade`: `null`
- Database query: All actors ordered by birthdate ASC

### With Valid Year Parameter

```
GET /actorsByDecade/1970
```

- Returns actors born between 1970 and 1979
- Title: "Listado de actores nacidos en la década de 1970 a 1979"
- `selectedDecade`: `1970`
- Database query: `whereBetween('birthdate', [1970, 1979])`

### With Invalid Year Parameter

```
GET /actorsByDecade/1850  (too early)
GET /actorsByDecade/2030  (too late)
GET /actorsByDecade/abc   (not numeric)
```

- Redirects to home page (`/`)
- Logs error: "Invalid year range accessed in actor middleware: {year}"
- No view is rendered

## Decade Range Calculation

When you provide a year, the endpoint includes actors born in a 10-year range:

- Year 1950 → Actors born 1950-1959
- Year 1960 → Actors born 1960-1969
- Year 1970 → Actors born 1970-1979
- Year 1980 → Actors born 1980-1989
- Year 1990 → Actors born 1990-1999
- Year 2000 → Actors born 2000-2009

<Note>
  The endpoint calculates the end year as `$year + 9`, creating an inclusive 10-year range.
</Note>
