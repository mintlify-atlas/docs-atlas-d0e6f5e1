## Overview

This endpoint returns all actors from the database, ordered chronologically by birthdate (ascending). It provides a comprehensive list with full actor details and available decades for filtering.

## Endpoint

```http
GET /allActors
```

## Middleware

This endpoint is protected by the `actor` middleware (`ValidateURLActor`).

## Request

No parameters required for this endpoint.

### Headers

No special headers required.

## Implementation Details

### Controller Method

Location: `app/Http/Controllers/ActorController.php:30`

```php
public function listAllActors()
{
    $title = "Listado de todos los actores";
   
    $actors = ActorController::readActors();

    if (empty($actors)) {
        $title = "No hay actores disponibles";
        return view("actors.list", [
            "actors" => [],
            "title" => $title
        ]);
    }

    return view("actors.list", [
        "actors" => $actors,
        "title" => $title,
        "decades" => ActorController::getAvailableDecades(),
        "selectedDecade" => null
    ]);
}
```

### Database Query

The `readActors()` method executes:

```php
Actor::orderBy('birthdate', 'asc')->get()->toArray()
```

### Route Definition

Location: `routes/web.php:59`

```php

    Route::get('allActors', [ActorController::class, "listAllActors"])

```

## Response Behavior

### Empty Database

When no actors exist in the database:
- Returns empty array for `actors`
- Title changes to: "No hay actores disponibles" (No actors available)
- `decades` field is not included
- `selectedDecade` field is not included

### Normal Response

When actors exist:
- All actors ordered by `birthdate` ASC
- Includes complete actor information
- Provides `decades` array for decade-based filtering
- `selectedDecade` is `null` (no filter applied)
