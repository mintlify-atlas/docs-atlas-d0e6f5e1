## Overview

This endpoint returns all films from the database sorted by release year in descending order (newest to oldest). This is useful for displaying films chronologically with the most recent releases first.

## Endpoint

```
GET /filmout/sortFilms
```

## Path Parameters

This endpoint does not accept any path parameters.

## Middleware

<Note>
  This endpoint is protected by the `year` middleware group, though it does not use any year parameters.
</Note>

## Implementation Details

- **Controller Method**: `FilmController::sortFilms()`
- **View**: `films.list`
- **Sorting Logic**: Uses PHP's `usort()` with spaceship operator (`<=>`) to sort by year in descending order

## Sorting Algorithm

The endpoint uses the following comparison function:

```php
usort($films, function($a, $b) {
    return $b['year'] <=> $a['year'];
});
```

This ensures films are ordered from **newest to oldest** (2024 → 1900).

<Note>
  The spaceship operator (`<=>`) returns:
  - `-1` if left is less than right
  - `0` if they are equal
  - `1` if left is greater than right
  
  By using `$b <=> $a` instead of `$a <=> $b`, the sort is in descending order.
</Note>
