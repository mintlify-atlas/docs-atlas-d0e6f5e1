# Laravel Films & Actors

A Laravel-based web application for managing and browsing films and actors with advanced filtering, sorting, and database capabilities.

## Overview

Laravel Films & Actors is a full-stack web application built with Laravel 10 that provides a comprehensive platform for managing film and actor data. The application combines JSON-based storage with database-driven architecture to offer flexible data management and rich filtering capabilities.

## What's Inside

This application demonstrates modern Laravel development practices including:

- **RESTful Routing** - Clean, organized route structure with prefix groups and middleware
- **Custom Middleware** - Year validation and URL parameter validation
- **Eloquent ORM** - Database models with relationships and query builders
- **Blade Templates** - Dynamic views for films, actors, and listings
- **Database Migrations** - Version-controlled schema management
- **Seeding & Factories** - Automated test data generation

## Use Cases

Laravel Films & Actors is ideal for:

- Learning Laravel development patterns and best practices
- Building film and media catalog applications
- Understanding middleware implementation in Laravel
- Exploring database relationships and migrations
- Creating content filtering and sorting systems
